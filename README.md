# NomadFlow

NomadFlow is a modern, Tesla-style Vietnam budget planner for digital nomads. It helps you translate foreign savings into VND and understand how long your budget can sustain a comfortable life in Vietnam.

## Key Features
- Real-time currency conversion to VND using live exchange rates
- Regional cost-of-living presets for quick monthly budget estimates
- Visual budget safety buffer that reflects runway at a glance

## Tech Stack
- HTML5
- CSS3 (modern UI/UX principles)
- Vanilla JavaScript (async/fetch for exchange rates)

## How to Use
1. Select your currency.
2. Enter your total savings.
3. Set monthly spending manually or click a regional preset.
4. Click "Analyze Budget" to see the VND equivalent and longevity.
